# Blinkit-Sales-Dashboard
📊 Blinkit Sales Analysis Dashboard | Power BI Project
Excited to share my latest Power BI dashboard project on Blinkit Sales Analysis, where I transformed raw retail data into actionable business insights.
🔍 Key Insights from the Dashboard:
✅ Total Sales: ₹997.16K
✅ Average Sales: ₹141.24
✅ Total Items Sold: 7K
✅ Average Rating: 3.92 ⭐
📈 Dashboard Highlights:
🔹 Sales performance analysis across outlet establishment years.
🔹 Comparison of Low Fat vs Regular products.
🔹 Item category-wise sales and average sales tracking.
🔹 Interactive filters for Item Identifier, Item Type, Fat Content, and Outlet Location.
🔹 KPI cards for quick business performance monitoring.
🔹 Dynamic visualizations for better decision-making.
